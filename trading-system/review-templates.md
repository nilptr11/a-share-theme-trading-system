# 复盘模板

> 用于每日、每周、每 15 笔或每月复盘。

## 1. 每日复盘

```text
日期：________  市场状态（A/B/C/D）：________
市场环境评分（/10）：指数__/2 + 成交量__/2 + 短线情绪__/3 + 主线清晰度__/3 = __/10
全市场成交额（亿）：____  涨停/跌停（排除ST）：____/____
连板高度：____  炸板率：____%  最强板块：________  最强个股：________
逻辑级别：国家大势 / 政策支持 / 技术进步 / 个股利好
资金状态：放量 / 缩量 / 分歧 / 回流
情绪状态：扩散 / 高潮 / 分化 / 退潮

今日交易：
买入标的：________  买入理由：________
逻辑/资金/情绪共振：是/否  位置赔率：止损__% 目标__% 赔率__
买点类型：7.1 低位突破 / 7.2 缩量回踩 / 7.3 分歧反包
买入价/止损价：________/________  风险预算/仓位（股数+金额）：________
卖出价：________  卖出理由：________  盈亏（含成本）：________

是否遵守系统：是/否  错误类型：________  明日计划：________
```

## 2. 每周复盘（差异项）

在每日复盘基础上，汇总：

```text
本周交易次数 / 胜率 / 盈亏比 / 最大单笔盈亏 / 周盈亏合计（扣成本）
最大回撤 / 最赚钱买点类型 / 最亏钱买点类型
是否有冲动交易 / 违反止损 / 过度交易 → 下周改进重点
```

## 3. 每 15 笔 / 每月统计分析

```text
1. 三种买点类型的胜率和平均盈亏比对比
2. 不同市场评分（6-7 vs 8-10）下的交易表现
3. 不同仓位下的盈亏分布
4. 不同时段表现（早盘 vs 尾盘，周一 vs 周五）
5. 最大 3 笔盈利和最大 3 笔亏损的共同特征
6. 违反规则的交易占比及盈亏
7. 交易成本合计，占盈亏的比例

决策规则：
- 某类买点连续 5 笔胜率 < 35% → 暂停该类买点
- 违反规则的交易平均亏损 > 遵守规则的 → 纪律是核心问题
- 交易成本占盈利 > 20% → 频率过高，降低
- 某市场状态下胜率显著偏低 → 该状态下禁止交易
```

复盘回答三个问题：

```text
1. 有没有按系统交易？
2. 系统本身有没有正期望？
3. 亏损是正常试错，还是违反规则？
```
