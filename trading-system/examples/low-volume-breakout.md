# 完整交易案例：低位放量突破

### 背景

```text
市场状态：A（强势）  市场评分：8 分
  指数环境：2 分（站上 20 日均线且向上）
  成交量环境：2 分（全市场成交额 ≥ 5 日均量 1.2 倍）
  短线情绪：2 分（4 项达标）
  主线清晰度：2 分（AI 板块成交占比 13%）
成交量连续 3 日放大，涨停 45 只，连板高度 4 板，炸板率 22%
主线：AI 板块持续走强
```

### 选股与评分

```text
个股 A 属于 AI 板块
逻辑级别：政策支持 + 技术进步（2 级）
资金状态：板块成交额连续 3 日站上 5 日均量 1.3 倍，个股成交额进入板块前 5
情绪状态：板块涨停数从 3 只增至 7 只，核心股带动后排扩散
流动性：日均成交额 1.2 亿，流通市值 80 亿 ✓
过去 10 个交易日横盘，今日放量（5 日均量 1.6 倍）突破平台高点（创近 20 日新高）

个股评分：成交量 2 + 相对强度 2 + 趋势位置 3 + 市场环境 1 = 8 分 ✓
```

### 买入计划

```text
买点类型：7.1 低位放量突破
风险预算：200 元（本金 2 万 × 1%）
买入价：20 元 / 止损价：19.2 元（跌破平台顶部下方）
止损空间：0.8 元/股（-4%）
仓位：200÷0.8 = 250 股 × 20 元 = 5000 元（25% 仓位）
最大亏损：250 × 0.8 = 200 元 = 本金的 1% ✓
```

### 买入后处理

```text
情况一：次日继续放量上涨 → 持有
情况二：次日缩量回踩不破突破位 → 观察
情况三：次日低开低走收盘 < 19.2 元 → 止损
情况四：板块整体退潮（跌幅 ≥ 3%）→ 主动减仓
```

### 卖出计划

```text
趋势强 → 持有到跌破 5 日线或放量滞涨
板块高潮（涨停 ≥ 8 只）→ 兑现 50%
技术破位（收盘 < 19.2 元）→ 全部卖出
```

### 复盘检查

```text
买入是否符合系统？止损是否提前设定？仓位是否按公式计算？
卖出是否按规则执行？这笔交易能否重复？
```
