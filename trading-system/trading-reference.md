# 交易规则库

> 用于查询规则、评分模型、买卖点定义、仓位与风控标准。实际执行时优先使用 [trading-playbook.md](trading-playbook.md)。

## 1. 适用市场环境

这套模式更适合：

```text
强势市场 / 结构性行情 / 题材活跃期 / 成交量放大期 / 短线赚钱效应较强阶段
```

不适合：

```text
缩量阴跌 / 题材一日游 / 连板高度下降 / 炸板率高（≥ 40%）/ 跌停数量增加 / 指数放量破位 / 主线持续退潮
```

第一原则：

```text
先判断市场能不能做，再判断个股值不值得做。
```

## 2. 市场环境评分与状态识别

每天交易前先给市场打分，满分 10 分。

### 2.1 指数环境：0-2 分

```text
2分：沪指/创业板指站上 20 日均线，且 20 日均线方向向上
1分：指数在 20 日均线 ±1% 以内连续 3 日震荡，均线走平
0分：指数在 20 日均线下方，或 20 日均线方向向下
```

### 2.2 成交量环境：0-2 分

```text
2分：全市场成交额 ≥ 近 5 日均值的 1.2 倍
1分：全市场成交额在近 5 日均值的 0.85-1.2 倍之间
0分：全市场成交额 < 近 5 日均值的 0.85 倍
```

### 2.3 短线情绪：0-3 分

观察以下 6 个指标，每个满足得 0.5 分，合计四舍五入：

```text
a. 昨日涨停指数今日收阳
b. 市场最高连板 ≥ 3 板
c. 跌停数 ≤ 5 只（排除 ST）
d. 炸板率 ≤ 30%
e. 昨日涨停股今日平均涨幅 ≥ 1%
f. 跌幅 > 7% 的个股 ≤ 30 只
```

映射关系：

```text
总分 ≥ 2.5 → 3分：连板高度打开，强势股持续赚钱
总分 ≥ 1.5 → 2分：赚钱效应较好
总分 ≥ 0.5 → 1分：情绪弱修复
总分 < 0.5 → 0分：跌停多、炸板高、高标杀跌
```

### 2.4 主线清晰度：0-3 分

```text
3分：存在单一板块，成交额占全市场 ≥ 15%，且连续 3 日走强
2分：存在 1-2 个主线板块，成交额占比 10-15%，持续 2 日以上
1分：多个题材轮动，没有单一板块持续 2 日以上
0分：没有主线，资金乱打，板块一日游
```

### 2.5 市场结论

```text
8-10分：可积极交易
6-7分：  轻仓试错
4-5分：  谨慎，只做最强
0-3分：  空仓或极低仓
```

### 2.6 市场状态识别（每周评估一次）

在评分之上，加一层更高的判断——当前市场处于什么状态。

| | 状态 A | 状态 B | 状态 C | 状态 D |
|------|------|------|------|------|
| 特征 | 强势/题材活跃 | 震荡/结构性 | 弱势/缩量阴跌 | 极端波动 |
| 市场评分 | ≥ 7 分持续 | 4-6 分波动 | ≤ 4 分持续 | 不适用 |
| 主线 | 明确且持续 | 轮动快 | 无 | 混乱 |
| 成交量 | 放大 | 平稳 | 萎缩 | 异常放大/缩小 |

每周复盘第一行写当前市场状态。状态 C/D 期间禁止实盘交易。

### 2.7 各状态应对规则

```text
状态 A（强势）：
  正常交易，标准仓位，三种买点均可使用，趋势持有可适度延长

状态 B（震荡）：
  风险预算降至 0.5%，禁用追涨买点（6.3），只做 6.1 和 6.2
  盈利预期减半，更快止盈（高潮兑现条件降一档）

状态 C（弱势）：
  完全空仓，只做观察和模拟记录
  记录"如果按系统做，哪只股票会是买点"——验证系统在熊市中的选股质量

状态 D（极端）：
  空仓，不猜底不抄底
  等待至少 2 个交易日，市场评分恢复到 ≥ 5 分再评估
```

### 2.8 状态切换规则

```text
状态切换需要连续 3 日确认，不因单日波动改变状态
状态 A → B：市场评分连续 3 日 ≤ 6 分
状态 B → A：市场评分连续 3 日 ≥ 7 分 + 主线持续 2 日
状态 B → C：市场评分连续 3 日 ≤ 4 分
状态 C → B：市场评分连续 3 日 ≥ 5 分 + 出现持续 2 日的主线
任何状态 → D：指数单日波动 ≥ 3%
状态 D → 其他：等待 2 个交易日后重新评估
```

## 3. 逻辑-资金-情绪-位置四位一体

选股不能只看图形，也不能只看消息。真正值得参与的机会，来自四类共振。

### 3.1 逻辑共振：定方向

重点看：国家大势、政策支持、产业趋势、业绩兑现、技术进步、事件催化。

题材优先级参考：

```text
1级：国家大势 > 2级：政策支持 > 3级：技术进步 > 4级：个股利好
```

但短线不能只看题材级别，还要看资金反应：

```text
长期看题材级别，短期看资金反应。
```

真正有效的排序应是：

```text
题材级别 × 资金强度 × 市场阶段 × 个股地位
```

**量化辅助：**
- 政策级别：国家级/部委/地方政府/行业协会 → 3/2/1/0 分
- 板块内涨停数 ≥ 5 只 → 题材有板块效应
- 板块内出现 3 连板以上个股 → 题材有高度标杆

### 3.2 资金共振：验真伪

不要过度依赖软件上的主力净流入，更要看真实成交结果。

优先观察：

```text
板块成交额是否持续放大 / 核心股成交额是否进入市场前排
上涨是否放量 / 回调是否缩量 / 分歧后是否快速修复 / 板块内是否多个个股联动
```

**量化锚点：**

| 信号 | 标准 |
|------|------|
| 板块成交额持续放大 | 连续 2 日 ≥ 板块 5 日均量的 1.3 倍 |
| 核心股成交额前排 | 进入全市场前 50 或板块前 5 |
| 上涨放量 | 上涨日成交额 ≥ 5 日均量的 1.2 倍 |
| 回调缩量 | 回调日成交额 ≤ 前一日成交额的 70% |
| 分歧后修复 | 分歧后次日或次次日收复分歧日跌幅的 60% 以上 |

资金共振成立时，通常不是单只股票独涨，而是：

```text
龙头走强 → 中军放量 → 后排扩散 → 板块成交额放大 → 分歧后资金回流
```

### 3.3 情绪共振：提弹性

重点看：题材是否契合市场主旋律、涨停数量是否增加、连板高度是否打开、板块是否集体异动、后排是否扩散、炸板率是否可控。

**量化锚点：**
- 板块内涨停数较前日增加 ≥ 3 只 → 情绪扩散
- 炸板率 ≤ 25% → 情绪可控
- 连板高度 ≥ 4 板 → 短线情绪较热
- 板块涨幅 ≥ 3% 且成交额创近 5 日新高 → 板块高潮信号

情绪共振越强，短线弹性越大；但情绪高潮后，也越容易分化。

### 3.4 四位一体判断

理想机会是：

```text
逻辑有级别（题材级别 ≥ 2 级）
资金有验证（板块成交额持续放大 + 核心股前排）
情绪有扩散（涨停数增加 + 后排联动）
位置有赔率（止损空间 ≤ 本金的 1%，上涨空间 ≥ 止损空间的 2 倍）
风险可控制（有明确止损位）
```

如果只有逻辑没有资金，不买；只有资金没有逻辑，谨慎；只有情绪没有位置，容易追高。

## 4. 选股池规则

不全市场乱看，只关注逻辑清晰、资金集中、情绪活跃的地方。

筛选分三层：

```text
底层：契合大势与主线，匹配全年或阶段性市场旋律
中层：资金认可 + 题材容量，能容纳持续资金进出
表层：多逻辑共振 + 高弹性，至少具备 2 个核心催化点
```

交易化筛选顺序：

```text
1. 方向是不是市场主线？
2. 板块有没有成交额和持续性？
3. 个股是不是板块核心？
4. 当前位置有没有赔率？
5. 错了能不能快速退出？
```

只有前三条成立，是选股；五条都成立，才是交易机会。

### 4.1 板块核心股

优先看：板块成交额前排、板块涨幅前排、最先启动、回调最抗跌、反弹最主动、能带动板块情绪。

**量化锚点：** 个股涨幅连续 3 日排名板块前 20%，计为核心股。

### 4.2 成交额前排股

重点观察：全市场成交额前 50、板块内成交额前 5、当日成交额显著放大的个股。成交额太小的个股，即使涨得快，也容易出现流动性风险（见 5.4 节）。

### 4.3 创新高或接近新高的强势股

强势股常见特征：

```text
不断新高 / 回调不深（最大回撤 ≤ 涨幅的 38.2%）/ 修复很快（分歧后 2 日内收复 60%+）
大盘调整时抗跌 / 板块修复时领涨
```

不要天然害怕新高。短线里，新高有时不是风险，而是强势确认。

### 4.4 流动性底线

以下条件不满足，不参与：

```text
个股日均成交额 ≥ 5000 万（验证期）
个股日均换手率 ≥ 2%
流通市值 ≥ 20 亿（验证期）
```

A 股 T+1 制度下，极端行情跌停板可能连续多日无法成交，理论"止损 3%"可能变成实际亏损 20%+。不参与流动性边缘的个股，不在市场恐慌期重仓。

### 4.5 主线第一次分歧后的修复股

条件：

```text
板块主线没有结束
龙头没有彻底走坏（未跌破 10 日线或前高平台）
第一次明显分歧
分歧日成交额 ≤ 前一日 70%（缩量分歧）
分歧后资金快速回流（次日或次次日收复分歧日跌幅 60%+）
核心股率先修复
```

## 5. 个股评分模型

满分 10 分。

```text
成交量：  0-3分
相对强度：0-3分
趋势位置：0-3分
市场环境：0-1分
```

### 5.1 成交量（0-3分）

```text
3分：当日成交额 ≥ 近 5 日均量的 2 倍，且收阳
2分：当日成交额 ≥ 近 5 日均量的 1.5 倍，且收阳
1分：当日成交额在近 5 日均量的 1.0-1.5 倍之间
0分：当日成交额 < 近 5 日均量的 1.0 倍，或放量收阴
```

### 5.2 相对强度（0-3分）

取近 5 个交易日，统计个股涨幅在所属板块内的百分位排名。

```text
3分：近 5 日有 ≥ 3 日涨幅排进板块前 20%，且板块修复时领涨
2分：近 5 日有 ≥ 2 日涨幅排进板块前 20%，强于板块平均
1分：近 5 日涨幅大部分在板块中位附近，跟随板块
0分：近 5 日弱于板块平均，或板块涨它不涨
```

辅助观察：大盘跌它不跌、板块回调它抗跌（回调幅度 ≤ 板块回调幅度的 70%）、板块反弹它先涨。

### 5.3 趋势位置（0-3分）

```text
3分：股价在 5/10/20 日均线之上，三线多头排列，且距近 20 日高点 < 5%
2分：股价在 5/10 日均线之上，20 日均线走平或向上，趋势健康
1分：股价在均线附近纠缠，均线走平，方向不明
0分：股价在均线下方，均线空头排列，或距近 20 日高点 > 15%
```

### 5.4 市场环境加权（0-1分）

```text
1分：当日市场评分 ≥ 8 分
0分：当日市场评分 < 8 分
```

### 5.5 交易门槛

```text
5分以下：不做
6分：     只观察或极小仓试错
7分以上： 可考虑交易
8-9分：   重点关注
10分：    极少出现，仍需按风控执行
```

## 6. 买点模型

三类介入方式：

```text
埋伏低吸：题材未完全启动，赔率高但确定性低，不可重仓
追涨买入：题材共振明确，确定性更高，但不买后排
回踩加仓：核心逻辑未破，只在重新转强后加仓，不能做成下跌补仓
```

核心原则：

```text
买点必须同时回答：为什么是它、为什么是现在、错了在哪里走。
```

**总规则：** 不属于 6.1/6.2/6.3 定义的标准买点 → 验证期内禁止参与。验证期通过后，每新增一种买点类型，必须先完成 10 笔小仓测试。

### 6.1 买点一：低位放量突破

适合题材刚启动。

**条件：**

```text
1. 个股经过至少 5 个交易日横盘整理
2. 题材已有预热或板块已有异动迹象
3. 突破近期平台高点（创近 20 日新高）
4. 当日成交额 ≥ 5 日均量的 1.5 倍
5. 收盘站上突破位（收盘价 ≥ 突破位 × 1.005）
6. 板块同步走强（板块涨幅 ≥ 1%）
7. 大盘环境不差（市场评分 ≥ 4 分）
```

**硬边界：** 突破当日必须创近 20 日新高，否则不算突破。

买入方式：

```text
突破确认时小仓试错（计划仓位的 50%）
收盘站稳可加至计划仓位
次日强势确认（收阳且量能不萎缩 30% 以上）再加剩余
```

失败信号与处理：

```text
突破后第二天低开低走 / 跌回平台内部（收盘价 < 突破位 × 0.99）
成交额放大但价格走弱（放量长上影或收阴）/ 板块没有跟随

→ 突破失败要快速止损，不补仓摊低成本。
```

---

### 6.2 买点二：主升第一次缩量回踩

适合已经走强的趋势股。

**条件：**

```text
1. 个股已经确认主升趋势（连续 3 日在 5 日线上方，且 5 日线向上）
2. 第一次明显回调（主升以来首次回踩短期均线）
3. 回调缩量（回调日成交额 ≤ 前一日成交额的 70%）
4. 不破 5 日线/10 日线/前高平台（收盘价 ≥ 支撑位）
5. 板块没有退潮（板块成交额未萎缩至 5 日均量以下）
6. 次日重新转强（收阳且成交额 ≥ 回调日成交额的 1.2 倍）
```

**硬边界：** 必须是主升以来的"第一次"回踩。第二次及以后的回踩不属于此模型——往往不是机会，而是趋势走弱的开始。

买入方式：

```text
回踩支撑附近观察 → 重新放量转强时介入 → 不在下跌惯性中盲目接
```

核心原则：**买转强，不买下跌惯性。**

失败信号：

```text
缩量回踩变成放量下跌（回调日成交额 ≥ 前一日 90% 且收中阴）
跌破关键支撑（收盘价 < 支撑位 × 0.99）
反弹无量（反弹日成交额 < 5 日均量的 90%）
板块核心股走坏
```

---

### 6.3 买点三：强势分歧后的反包

适合短线情绪强的阶段。

**条件：**

```text
1. 个股是板块核心（近 5 日涨幅排板块前 20%）
2. 前一日出现分歧（炸板、断板、冲高回落，收长上影或假阳线）
3. 次日没有继续走弱（平开或高开，非低开低走）
4. 开盘或盘中快速转强（前 30 分钟内翻红或维持红盘）
5. 当日成交额 ≥ 5 日均量（能承接抛压）
6. 板块其他核心股配合（板块内至少 2 只个股同步走强）
```

**硬边界：** 分歧次日必须高开或平开快速翻红。低开低走后盘中拉回的不算反包——属于弱势挣扎，成功概率大幅降低。

买入方式：

```text
反包确认时介入（涨幅超过分歧日收盘价 1% 且量能配合）
不能提前赌（不在分歧日尾盘埋伏）
```

失败信号：

```text
高开低走（开盘后 30 分钟内翻绿且持续走弱）
反包失败（盘中冲高但收盘回落，涨幅 < 分歧日跌幅的 50%）
放量不能上涨（成交额大但价格横盘或微涨）
板块分歧扩大（板块内出现跌停股或核心股走弱）/ 龙头跌停
```

## 7. 卖点模型

卖点优先级高于买点。买错不可怕，错了不卖才危险。

卖出信号分四类：

```text
封单异常：涨停承接变弱，可能是炸板或分歧扩大前兆
走势不及预期：该强不强，板块涨它不涨，突破后跌回平台
资金分歧：放量不能上涨，核心股承接变差，板块回流失败
逻辑破位：题材降温、事件落地、政策或业绩不及预期
```

不要只等逻辑破位才卖——价格、资金和情绪经常更早反映变化。

卖出判断使用组合原则：

```text
逻辑破、资金弱、技术破、情绪退，任意两个出现就减仓或卖出；
触发硬止损，必须卖出。
```

### 7.1 止损卖点

**固定亏损止损：**

```text
单笔亏损达到总本金的 1%-3%，必须处理。
```

**技术破位止损：**

```text
跌破突破位（收盘价 < 突破位 × 0.99）/ 跌破前低
跌破 5 日线/10 日线（收盘确认）
放量长阴（跌幅 ≥ 5% 且成交额 ≥ 5 日均量 1.5 倍）
跌回平台内部 / 反包失败（买入后次日不能继续走强）
```

**市场环境止损：**

即使个股没怎么跌，如果市场环境突然恶化，也要减仓或退出：

```text
高标集体杀跌 / 主线龙头跌停
指数放量破位（单日跌 ≥ 2% 且成交额 ≥ 5 日均量 1.3 倍）
跌停数量明显增加（较前日增加 ≥ 50%）
板块大面积退潮（板块跌幅 ≥ 3%）
```

---

### 7.2 止盈卖点

**趋势持有条件：**

```text
放量上涨（上涨日成交额 ≥ 5 日均量）+ 缩量回调（回调日成交额 ≤ 前一日 70%）
不破趋势（收盘在 5 日线上方）+ 板块仍强 + 核心股没有明显走弱
```

**高潮兑现信号：**

```text
连续大涨（3 日累计涨幅 ≥ 20%）
板块集体高潮（板块内涨停 ≥ 8 只，或板块涨幅 ≥ 4%）
后排补涨 / 媒体大量讨论
高位爆量滞涨（成交额 ≥ 5 日均量 2 倍但涨幅 < 1%）
封单快速减少 / 大单频繁撤单 / 涨停反复打开
```

**走弱卖出信号：**

```text
上涨无量（上涨日成交额 < 5 日均量的 90%）
冲高回落（长上影 ≥ 实体 2 倍）
跌破 5 日线（收盘确认）/ 反弹不过前高
板块回流它不涨 / 强势股变成跟风股
```

原则：**强势股一旦不强，就不能继续用强势股逻辑持有。**

### 7.3 流动性退出规则

当常规止损无法执行时，启用以下规则：

```text
持仓股跌停封单 ≥ 流通盘的 2% → 次日开盘直接挂跌停价卖出
连续 2 日无法在预设止损位成交 → 改为次日开盘价无条件出清
持仓股被公告 ST/*ST → 次日集合竞价无条件卖出
市场跌停数 ≥ 50 只 → 所有仓位降至验证期水平或以下
指数单日跌幅 ≥ 3% 且放量（成交额 ≥ 5 日均量 1.3 倍）→ 收盘前减仓至 50% 以下
```

## 8. 仓位规则

采用**风险预算制**。不同波动率的个股承担相同金额的风险。

### 8.1 风险预算公式

```text
每笔交易的风险预算 = 本金 × 风险比例（验证期 1%，标准期 1-2%）

仓位 = 风险预算 ÷ (买入价 - 止损价)

股数 = 仓位 ÷ 买入价（取整百股）
```

**举例（本金 2 万，验证期风险预算 1% = 200 元）：**

| 买入价 | 止损价 | 止损幅度 | 仓位金额 | 仓位比例 |
|--------|--------|----------|----------|----------|
| 20 元 | 18.5 元 | -7.5% | 200÷1.5 = 133 股 → 2660 元 | 13.3% |
| 20 元 | 19.5 元 | -2.5% | 200÷0.5 = 400 股 → 8000 元 | 40% |
| 10 元 | 9.4 元 | -6% | 200÷0.6 = 333 股 → 3330 元 | 16.7% |

波动大的股票自动获得更少仓位，波动小的获得更多。仓位由止损位和市场结构倒推，不是拍脑袋决定。

### 8.2 初始阶段：2 万以内

```text
风险预算：每笔 1%（200 元）
单票仓位上限：40%
最多同时 2-3 只
```

### 8.3 分阶段仓位管理

```text
2万-5万：   风险预算 1%，练系统，轻仓验证
5万-10万：  风险预算 1-1.5%，适度放大，控制回撤
10万-30万： 风险预算 1-2%，重视稳定性
30万以上：  风险预算 ≤ 2%，优先保住利润
```

资金越大，越要重视回撤控制。

## 9. 空仓规则

市场状态识别见第 2.6-2.8 节。以下为空仓触发条件：

```text
1. 市场成交量持续萎缩（全市场成交额连续 3 日 < 5 日均量的 0.85 倍）
2. 指数放量破位
3. 连板高度下降（最高连板 < 3 板）
4. 高标股连续被核
5. 跌停数量明显增加（较前日 ≥ 50%）
6. 主线题材退潮（板块成交额萎缩至高峰期的 60% 以下）
7. 板块轮动太快，一日游严重
8. 自己连续亏损 5 笔以上
9. 情绪失控，想报复交易
10. 看不懂市场主线
```

重要原则：

```text
看不懂的时候，不交易就是交易。
```

## 10. 小仓验证期执行规则

验证期不直接以盈利倍数为目标，而是先验证系统在自己手里是否具备正期望。

### 10.1 验证周期

```text
至少 30 笔交易，或连续 1-3 个月，二者满足其一后做阶段评估
```

验证期目标不是赚最多，而是确认：规则能不能执行、亏损能不能控制、哪类买点最有效、系统是否有正期望。

### 10.2 验证期约束

```text
风险预算：每笔本金 1%
单票仓位上限：40%
最多同时持有 2 只
单笔最大亏损：200 元以内（以 2 万本金计算）
不做满仓单吊，不因连续盈利临时放大仓位
仓位计算：仓位 = 风险预算 ÷ (买入价 - 止损价)，见第 8 节
```

### 10.3 交易门槛

验证期只做符合以下条件的交易：

```text
1. 市场评分 ≥ 6 分，市场状态为 A 或 B
2. 个股评分 ≥ 7 分
3. 逻辑、资金、情绪至少两项共振
4. 买点属于 6.1-6.3 定义的标准模型
5. 买入前已写清止损位
6. 单笔最大亏损不超过本金 1%
```

市场评分低于 6 分，只观察和记录，不做实盘验证。

### 10.4 暂停机制

```text
连续亏损 5 笔 → 暂停
单日亏损超过本金 2% → 暂停
账户从阶段高点回撤超过 10% → 暂停
出现一次不按止损执行 → 暂停
出现一次报复交易 → 暂停
连续 3 笔交易不符合文档规则 → 暂停
```

暂停后至少间隔 1 个完整交易日才能恢复，期间完成所有未记录的复盘并写下暂停原因。

### 10.5 验证通过标准

完成 30 笔交易或 1-3 个月后：

```text
胜率 ≥ 45%
盈亏比 ≥ 1.5
最大回撤 < 10%
规则执行率 ≥ 90%
没有单笔纪律性大亏
主要盈利来自标准买点（6.1/6.2/6.3），而非偶然暴利
```

不满足则不扩大仓位，只优化规则或继续小仓验证。

### 10.6 记录要求

每笔交易必须记录：

```text
市场评分 / 市场状态 / 逻辑级别 / 资金状态 / 情绪状态
个股评分（四项得分）/ 买点类型 / 买入价 / 止损价
仓位（股数+金额）/ 风险预算使用额
预设止损 / 实际卖点 / 盈亏金额+比例（含交易成本）
是否遵守系统 / 错误类型
```

没有记录的交易，视为无效样本。

## 11. 目标管理与回撤控制

### 11.1 三级递进目标

放弃固定倍数目标，改为验证自己能稳定赚多少。

**第一级：生存目标（前 6 个月）**

```text
✓ 本金不出现 20% 以上回撤
✓ 完成 30 笔标准交易
✓ 规则执行率 ≥ 80%
✓ 建立完整的复盘数据库
```

此阶段**不设盈利目标**。活下来、有数据，就是成功。

**第二级：验证目标（6-12 个月）**

```text
✓ 年化收益 20-50%（专业级水平）
✓ 最大回撤 < 15%
✓ 胜率 × 盈亏比 > 1.0（正期望确认）
✓ 确认最有效的买点类型
```

达标后用利润而非本金放大仓位。验证期月均收益 X%，后续按 X% 推演，不按预设神话。

**第三级：增长目标（12 个月以后）**

```text
基于验证期实际数据推演。月均 5% 按 5% 推演，月均 10% 按 10% 推演。
不强行追逐事先设定的倍数。
```

月均复利 17.7% 要求在胜率 45%、盈亏比 1.5 条件下每月完成约 47 笔有效交易——T+1 制度下几乎不可能。盯着倍数会诱导重仓和报复交易。真正的复利来自"少亏 + 稳赚 + 长时间"，不来自"某一笔大赚"。

### 11.2 回撤控制

```text
亏 10%，需要涨 11.1% 回本
亏 20%，需要涨 25% 回本
亏 30%，需要涨 42.9% 回本
亏 50%，需要涨 100% 回本
```

阶段性回撤标准：

```text
10% 以内：优秀
15% 以内：还能修复，但需降低仓位
20% 以上：复利节奏明显受损，暂停交易，复盘系统和执行问题
30% 以上：系统需要全面检查
```

账户从阶段高点回撤超过 10%，降低仓位至验证期水平；回撤超过 15%，暂停交易，全面复盘。

## 12. 交易心理规则

### 12.1 日常纪律

```text
当日亏损达到本金 1%，当天不再开新仓
连续盈利 5 笔后，下一笔风险预算降至标准的 50%
每笔交易前写："如果我错了，最大亏损是 ___ 元"，数字写下来再下单
不在情绪波动时做交易决策（愤怒、兴奋、焦虑、FOMO）
```

### 12.2 月度自检

```text
这个月有没有报复交易？有没有不按止损执行？
有没有盘中临时改仓位？有没有因为"这次不一样"违反规则？
有没有因为亏损而跳过复盘？
```

### 12.3 错误清单

记录的是违反规则的行为，不是亏了多少钱：

```text
- 盘中临时扩大仓位 / 下跌补仓摊成本 / 止损位到了但没执行
- 买点不符合标准模型 / 追高后排跟风股
- 因他人推荐买入 / 亏损后报复交易 / 盈利后自信心膨胀重仓
```

每次违反写在清单顶部。同一个错误出现 3 次，暂停交易 1 周。
