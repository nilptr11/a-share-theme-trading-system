# 交易执行手册

> 用来操作，不用来解释。执行时按清单逐项确认；需要查标准时回到 [trading-reference.md](trading-reference.md)。

## 1. 盘前清单

```text
[ ] 市场环境评分：__/10
[ ] 市场状态：A / B / C / D
[ ] 市场状态是否允许实盘交易？A/B 可评估，C/D 空仓
[ ] 当前主线板块：________
[ ] 主线逻辑级别：国家大势 / 政策支持 / 技术进步 / 个股利好
[ ] 板块成交额是否持续放大？连续 2 日 ≥ 板块 5 日均量 1.3 倍
[ ] 情绪是否扩散？板块涨停数较前日增加 ≥ 3 只
[ ] 是否找出板块核心个股？
[ ] 是否标记可能买点？规则库 6.1 / 6.2 / 6.3
[ ] 是否预设止损位置？
[ ] 是否用风险预算公式计算仓位？
```

## 2. 是否允许交易

```text
[ ] 市场评分 ≥ 6 分
[ ] 市场状态为 A 或 B
[ ] 个股评分 ≥ 7 分
[ ] 逻辑、资金、情绪至少两项共振
[ ] 买点属于标准模型：规则库 6.1 / 6.2 / 6.3
[ ] 买入前已写清止损位
[ ] 单笔最大亏损不超过本金 1%
```

任一不满足：只观察，不实盘。

## 3. 盘中观察

```text
[ ] 板块是否继续放量？
[ ] 核心股是否强于板块？
[ ] 题材情绪是扩散、高潮、分化还是退潮？
[ ] 是否出现标准买点？
[ ] 买入前是否确认止损空间？
[ ] 是否按公式计算仓位并写入交易记录？
[ ] 是否避免临时扩大仓位？
```

## 4. 买入前确认

```text
标的：________
买点类型：规则库 6.1 低位放量突破 / 规则库 6.2 主升第一次缩量回踩 / 规则库 6.3 强势分歧后的反包
个股评分：__/10
买入价：____
止损价：____
止损空间：____ 元/股，____%
本金：____
风险预算：本金 × ____% = ____ 元
仓位：风险预算 ÷ (买入价 - 止损价) = ____ 股
实际买入金额：____ 元
最大亏损：____ 元
```

买点必须同时回答：

```text
为什么是它？
为什么是现在？
错了在哪里走？
```

## 5. 买点执行清单

### 买点一：低位放量突破（规则库 6.1）

```text
[ ] 至少 5 个交易日横盘整理
[ ] 题材已有预热或板块已有异动迹象
[ ] 创近 20 日新高
[ ] 当日成交额 ≥ 5 日均量 1.5 倍
[ ] 收盘价 ≥ 突破位 × 1.005
[ ] 板块涨幅 ≥ 1%
[ ] 市场评分 ≥ 4 分
```

不创近 20 日新高，不算突破。

### 买点二：主升第一次缩量回踩（规则库 6.2）

```text
[ ] 连续 3 日在 5 日线上方，且 5 日线向上
[ ] 主升以来第一次明显回调
[ ] 回调日成交额 ≤ 前一日成交额 70%
[ ] 收盘价 ≥ 关键支撑位
[ ] 板块成交额未萎缩至 5 日均量以下
[ ] 次日收阳且成交额 ≥ 回调日成交额 1.2 倍
```

买转强，不买下跌惯性。

### 买点三：强势分歧后的反包（规则库 6.3）

```text
[ ] 个股近 5 日涨幅排板块前 20%
[ ] 前一日出现炸板、断板、冲高回落、长上影或假阳线
[ ] 次日平开或高开，非低开低走
[ ] 前 30 分钟内翻红或维持红盘
[ ] 当日成交额 ≥ 5 日均量
[ ] 板块内至少 2 只核心股同步走强
```

低开低走后盘中拉回，不算反包。

## 6. 买入后处理

```text
[ ] 符合预期：持有
[ ] 不符合预期：减仓或止损
[ ] 不因短期浮盈随意加仓
[ ] 不因短期浮亏随意补仓
```

## 7. 卖出触发清单

```text
[ ] 触发硬止损：必须卖出
[ ] 逻辑破、资金弱、技术破、情绪退，任意两个出现：减仓或卖出
[ ] 强势股不再强：不能继续用强势股逻辑持有
[ ] 市场环境突然恶化：即使个股没怎么跌，也要减仓或退出
```

## 8. 硬止损

```text
[ ] 单笔亏损达到总本金 1%-3%
[ ] 收盘价 < 突破位 × 0.99
[ ] 跌破前低
[ ] 跌破 5 日线/10 日线
[ ] 跌幅 ≥ 5% 且成交额 ≥ 5 日均量 1.5 倍
[ ] 反包失败，买入后次日不能继续走强
```

## 9. 止盈与兑现

```text
[ ] 3 日累计涨幅 ≥ 20%
[ ] 板块内涨停 ≥ 8 只，或板块涨幅 ≥ 4%
[ ] 后排补涨，媒体大量讨论
[ ] 成交额 ≥ 5 日均量 2 倍但涨幅 < 1%
[ ] 封单快速减少 / 大单频繁撤单 / 涨停反复打开
```

## 10. 暂停机制

```text
[ ] 连续亏损 5 笔 → 暂停
[ ] 单日亏损超过本金 2% → 暂停
[ ] 账户从阶段高点回撤超过 10% → 暂停
[ ] 出现一次不按止损执行 → 暂停
[ ] 出现一次报复交易 → 暂停
[ ] 连续 3 笔交易不符合规则 → 暂停
```

暂停后至少间隔 1 个完整交易日才能恢复，期间完成所有未记录的复盘并写下暂停原因。

## 11. 卖出后记录

```text
[ ] 买入理由
[ ] 卖出理由
[ ] 是否遵守纪律
[ ] 盈亏金额与比例，含交易成本
[ ] 错误类型
[ ] 这笔交易能否重复
```

## 12. 交易闭环

```text
判断市场状态（A/B/C/D）
        ↓
市场环境评分
        ↓
确定是否交易
        ↓
寻找主线板块 → 判断逻辑级别 → 确认资金共识 → 观察情绪扩散
        ↓
筛选核心个股（流动性底线 + 个股评分）
        ↓
等待标准买点（规则库 6.1/6.2/6.3）
        ↓
计算仓位（风险预算公式）
        ↓
按规则止损/止盈
        ↓
每日复盘 → 每 15 笔统计分析 → 迭代优化
```
