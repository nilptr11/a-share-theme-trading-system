# 个人交易 vs 量化交易的边界

本系统中的量化锚点是**人的辅助工具**，不是机器的执行指令。

## 1. 人可以做到、机器难以做到

| 能力 | 说明 |
|------|------|
| 解读政策意图 | 人能分辨"走过场"和"动真格"，机器只能做关键词匹配 |
| 感知情绪转折 | 封单从 10 万手缓慢减少 vs 瞬间撤单 8 万手，人能看到过程 |
| 识别异常情景 | "这个涨停和上个月那次一模一样，但当时板块已经退潮了" |
| 主动停止交易 | 连续盈利后主动降仓、感觉不对时空仓——机器不会"感觉不对" |
| 判断逻辑级别 | "国家大势" vs "政策支持"需要理解政策的真实影响范围 |
| 盘中临时判断 | 反包确认时的盘口细节、板块联动的微妙变化 |

## 2. 量化锚点的真正作用

```text
1. 防止事后合理化——"我当时觉得放量了" → 数字摆在那里
2. 统一判断尺度——今天的"放量"和上周的"放量"是同一个标准
3. 定位亏损原因——"这次亏损是因为成交额只到 1.3 倍而不是 1.5 倍就冲进去了"
4. 压缩灰色地带——把"差不多""感觉还行"压缩到具体数字区间
```

## 3. 如果要转为量化系统

```text
1. 消除所有定性描述——"板块核心股"需定义为具体排名规则，"逻辑级别"需 NLP 或人工标注
2. 回测数据至少覆盖一轮完整牛熊（A 股约 3-5 年），包含极端行情样本
3. 至少 3 个月纸交易/小资金测试滑点和冲击成本
4. 人工 override 机制——量化出信号，人做最终决策（过渡阶段保留人工确认）
   记录每次 override 的原因和结果
   当 override 频率 < 10% 且结果不比系统差时，才考虑全自动
```

## 4. 一句话

```text
本系统的量化是"带刻度的尺子"，不是"自动运行的机器"。
人用尺子量东西，尺子帮助人更准确；机器用尺子切割东西，还需要电机、导轨、安全开关——那是另一套系统。
```
