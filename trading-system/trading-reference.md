# 交易模式：核心执行版

> 目标：把交易系统压缩成盘前、盘中能直接执行的规则。本文档只保留核心思想、硬开关、关键数值和仓位公式。

---

## 0. 一句话模式

```text
只在市场允许时，做主线板块里的核心强势股，等待低位放量突破或主升第一次缩量回踩，用预设止损倒推仓位。
```

这套模式不是预测系统，而是筛选和风控系统：

```text
市场决定能不能做
主线决定看哪里
强弱决定选哪只
买点决定何时进
止损决定买多少
纪律决定能活多久
```

判断边界：

```text
人先判断方向，数字只做校准。
市场评分不是买入信号，只是交易权限校准。
主线判断不清楚，即使评分达标也不交易。
```

---

## 1. 最小执行流程

每天只按这个顺序判断：

```text
1. 今天能不能做？
2. 主线在哪里？
3. 谁是核心强势股？
4. 有没有标准买点？
5. 错了在哪里走？
6. 按止损距离可以买多少？
```

任一步回答不清楚：不交易。

机会成立条件：

```text
逻辑有级别
资金有确认
情绪有扩散
位置有赔率
风险可控制
```

```text
只有逻辑没有资金：不买。
只有资金没有逻辑：谨慎。
只有情绪没有位置：容易追高。
```

选股和交易机会分开：

```text
方向是主线、板块有成交额、个股是核心，只是进入观察池。
再满足位置有赔率、错了能快速退出，才是交易机会。
```

## 2. 三个交易开关

### 2.1 市场开关

只分三档，不再做复杂状态切换。

| 市场档位 | 条件 | 动作 | 风险预算 |
|----------|------|------|----------|
| 强 | 市场评分 ≥ 7 | 可交易 | 账户权益 1% |
| 中 | 市场评分 = 6 | 轻仓试错 | 账户权益 0.5% |
| 弱 | 市场评分 < 6 | 不开新仓 | 0 |

市场评分只保留 4 个维度，满分 10 分：

| 维度 | 分值 | 得分标准 |
|------|------|----------|
| 指数 | 0-2 | 指数站上 20 日线且 20 日线向上 = 2；震荡 = 1；跌破或向下 = 0 |
| 成交量 | 0-2 | 全市场成交额 ≥ 5 日均值 1.2 倍 = 2；0.85-1.2 倍 = 1；< 0.85 倍 = 0 |
| 情绪 | 0-3 | 涨停多、跌停少、炸板低、连板高度打开，综合给 0-3 |
| 主线 | 0-3 | 单一主线清晰且持续 = 3；1-2 条主线 = 2；轮动 = 1；无主线 = 0 |

硬规则：

```text
市场评分不是买入信号，评分达标后仍要等待主线、核心股和标准买点。
市场评分 < 6：不实盘开新仓。
连续 3 日全市场成交额 < 5 日均值 0.85 倍：空仓。
炸板率 ≥ 40%：不做追涨，只观察。
指数单日跌幅 ≥ 3%：停止新开仓。
看不懂主线：不交易。
```

---

### 2.2 主线开关

只有主线成立，才进入选股。

```text
题材先分级，资金再验证。
长期看题材级别，短期看资金反应。
```

优先级参考：

```text
国家大势 > 政策支持 > 产业趋势 / 技术进步 > 事件催化 > 个股利好
```

主线成立至少满足 3 条：

```text
1. 板块连续 2 日强于市场。
2. 板块成交额连续 2 日 ≥ 板块 5 日均量 1.3 倍。
3. 板块内有 ≥ 5 只涨停，或出现 3 连板以上高度股。
4. 板块内核心股成交额进入全市场前 50 或板块前 5。
5. 分歧后资金能回流，核心股没有集体走坏。
```

不做以下板块：

```text
一日游题材
只有消息没有成交额
只有后排补涨没有核心股带动
板块涨幅大但成交额没有放大
主线已经高潮后再扩散到低位杂毛
```

---

### 2.3 个股开关

个股必须是主线里的核心强势股，不做后排跟风。

核心强势股至少满足 3 条：

```text
1. 成交额进入板块前 5，或全市场前 50。
2. 近 5 日至少 2 日涨幅排名板块前 20%。
3. 板块分歧时更抗跌，板块修复时先反弹。
4. 股价在 5 日线、10 日线、20 日线之上，或正在突破近 20 日新高。
5. 能带动同板块个股，而不是被动跟涨。
```

```text
短线不天然害怕新高。
主线、资金、成交额同时确认的新高，是强势确认。
没有主线和资金的新高，才是追高风险。
```

流动性底线：

```text
日均成交额 ≥ 5000 万
日均换手率 ≥ 2%
流通市值 ≥ 20 亿
```

不满足流动性底线：不交易。

## 3. 只做两个买点

只保留两个买点。其他形态即使看起来机会很好，也先不做。

---

### 3.1 买点一：低位放量突破

适用场景：主线刚启动，个股从平台整理中走出来。

必须全部满足：

```text
1. 市场评分 ≥ 6。
2. 属于当前主线板块。
3. 个股经过至少 5 个交易日横盘整理。
4. 突破近 20 日高点。
5. 当日成交额 ≥ 5 日均量 1.5 倍。
6. 收盘价 ≥ 突破位 × 1.005。
7. 板块当日涨幅 ≥ 1%。
8. 买入前能写清楚止损位。
```

失败信号：

```text
收盘价 < 突破位 × 0.99
放量长上影或放量收阴
板块没有跟随
次日低开低走
```

处理：快速止损，不补仓。

---

### 3.2 买点二：主升第一次缩量回踩

适用场景：主线已经确认，核心股进入主升，第一次给低吸机会。

必须全部满足：

```text
1. 市场评分 ≥ 6。
2. 个股连续 3 日站上 5 日线，且 5 日线向上。
3. 这是主升以来第一次明显回踩。
4. 回调日成交额 ≤ 前一日成交额 70%。
5. 回调日成交额 ≤ 5 日均量 80%。
6. 收盘不破 5 日线、10 日线或前高平台。
7. 板块成交额没有跌破 5 日均量。
8. 次日收阳，且成交额 ≥ 回调日 1.2 倍。
```

核心原则：

```text
买转强，不买下跌惯性。
```

失败信号：

```text
回调变成放量下跌
收盘跌破关键支撑 × 0.99
反弹日成交额 < 5 日均量 90%
板块核心股集体走弱
```

处理：放弃或止损，不把回踩做成补仓。

## 4. 买入前必须回答三句话

任何一笔交易，下单前必须写清楚：

```text
为什么是它？
为什么是现在？
错了在哪里走？
```

标准答案示例：

```text
为什么是它：它是当前主线板块成交额前 5 的核心股，板块分歧时抗跌，修复时率先新高。
为什么是现在：今日放量突破近 20 日平台高点，成交额达到 5 日均量 1.6 倍，板块同步上涨 2%。
错了在哪里走：收盘跌回突破位 0.99 以下，或实际亏损达到本笔风险预算，卖出。
```

写不出来，不买。

## 5. 仓位规则

仓位由止损倒推，不由信心决定。

### 5.1 风险预算

```text
市场评分 ≥ 7：单笔最大计划亏损 = 账户权益 × 1%
市场评分 = 6：单笔最大计划亏损 = 账户权益 × 0.5%
市场评分 < 6：不交易
```

### 5.2 仓位公式

```text
风险预算金额 = 账户权益 × 风险预算比例
每股风险 = 买入价 - 止损价
计划买入股数 = 风险预算金额 ÷ 每股风险，向下取整百股
计划买入金额 = 计划买入股数 × 买入价
仓位比例 = 计划买入金额 ÷ 账户权益
```

硬限制：

```text
单票仓位 ≤ 40%
最多同时持有 2-3 只
不满仓单吊
不下跌补仓
不因连续盈利临时放大风险预算
加仓后总计划亏损仍不得超过本笔风险预算
```

### 5.3 交易成本

```text
买入前估算完整买卖成本。
预期盈利不足交易成本 10 倍，不参与。
复盘只看扣除交易成本后的净盈亏。
交易成本占毛盈利比例过高时，降低交易频率。
```

## 6. 卖出规则

卖出优先级高于买入。

### 6.1 可以继续持有

同时满足以下条件，可以继续持有：

```text
上涨放量，回调缩量。
收盘仍在 5 日线上方。
板块仍强。
核心股没有明显走弱。
没有触发硬止损或市场环境止损。
```

不满足持有条件时，不要用原来的买入理由继续持有。

### 6.2 必须卖出

出现任一条，必须处理：

```text
实际亏损达到本笔风险预算金额
收盘价 < 突破位 × 0.99
收盘跌破关键支撑位 × 0.99
收盘跌破 5 日线或 10 日线
跌幅 ≥ 5%，且成交额 ≥ 5 日均量 1.5 倍
买入后次日不能继续走强，反包或突破失败
```

市场突然恶化时，即使个股没有触发技术止损，也要减仓或退出：

```text
高标集体杀跌
主线龙头跌停
指数放量破位
跌停数量明显增加
板块跌幅 ≥ 3%
```

---

### 6.3 减仓或卖出

出现以下任意两个，减仓或卖出：

```text
逻辑弱：题材催化落地后没有资金继续承接
资金弱：放量但涨不动，成交额 ≥ 5 日均量 1.3 倍但涨幅 < 1%
技术弱：跌破 5 日线、反弹不过前高、长上影 ≥ 实体 2 倍
情绪退：板块涨停减少、炸板升高、核心股走弱
```

原则：

```text
强势股一旦不强，就不能继续用强势股逻辑持有。
```

---

### 6.4 主动止盈

出现以下信号，优先兑现一部分利润：

```text
个股 3 日累计涨幅 ≥ 20%
板块内涨停 ≥ 8 只
板块涨幅 ≥ 4%
板块涨幅 ≥ 3%，且成交额创近 5 日新高
成交额 ≥ 5 日均量 2 倍，但收盘涨幅 < 1%
后排补涨明显，媒体大量讨论
涨停反复打开，封单快速减少
```

### 6.5 流动性退出

常规止损无法成交时，不再等待理想价格：

```text
连续 2 日无法在止损位成交，次日开盘无条件出清。
持仓被 ST/*ST，次日集合竞价无条件卖出。
市场跌停数 ≥ 50，只降风险，不加仓、不抄底。
```

## 7. 空仓与暂停

以下情况，不开新仓：

```text
市场评分 < 6
看不懂主线
主线轮动太快，一日游严重
全市场成交额连续 3 日 < 5 日均量 0.85 倍
炸板率 ≥ 40%
指数放量破位
高标股集体杀跌
板块核心股集体走弱
自己情绪失控，想报复交易
```

日内刹车：

```text
当日亏损达到本金 1%，当天不再开新仓。
连续盈利 5 笔后，下一笔风险预算降至 0.5%。
```

以下情况，必须暂停交易：

```text
连续亏损 5 笔
单日亏损超过本金 2%
账户从阶段高点回撤 ≥ 10%
出现一次不按止损执行
出现一次报复交易
连续 3 笔交易不符合本规则
```

恢复规则：

```text
至少暂停 1 个完整交易日。
补齐复盘。
恢复后第一笔风险预算降至 0.5%。
```

## 9. 禁止清单

```text
不把市场评分当买入信号
不把选股当交易机会
不在市场评分 < 6 时开仓
不做非主线
不做后排跟风
不做没有流动性的票
不做第二次、第三次回踩
不做情绪高潮后的杂毛补涨
不提前赌反包
不下跌补仓
不临时扩大仓位
不把复盘理由当成盘中放宽规则的借口
```

## 10. 核心数值

| 类型 | 数值 |
|------|------|
| 可交易市场评分 | ≥ 6 |
| 标准风险预算 | 账户权益 1% |
| 降低风险预算 | 账户权益 0.5% |
| 日内停止开仓 | 当日亏损达到本金 1% |
| 暂停交易 | 单日亏损超过本金 2% |
| 连续盈利后降风险 | 连续盈利 5 笔后，下一笔风险预算 0.5% |
| 单票仓位上限 | 40% |
| 持仓数量 | 2-3 只以内 |
| 交易成本过滤 | 预期盈利 ≥ 完整交易成本 10 倍 |
| 市场放量 | 全市场成交额 ≥ 5 日均值 1.2 倍 |
| 市场缩量 | 全市场成交额 < 5 日均值 0.85 倍 |
| 板块持续放量 | 连续 2 日 ≥ 板块 5 日均量 1.3 倍 |
| 个股突破放量 | 当日成交额 ≥ 5 日均量 1.5 倍 |
| 强放量 | 当日成交额 ≥ 5 日均量 2 倍 |
| 突破确认 | 收盘价 ≥ 突破位 × 1.005 |
| 突破失败 | 收盘价 < 突破位 × 0.99 |
| 缩量回踩 | ≤ 前一日 70%，且 ≤ 5 日均量 80% |
| 回踩转强 | 次日成交额 ≥ 回调日 1.2 倍且收阳 |
| 放量长阴 | 跌幅 ≥ 5%，且成交额 ≥ 5 日均量 1.5 倍 |
| 板块高潮 | 涨停 ≥ 8 只，或板块涨幅 ≥ 4% |
| 个股短线高潮 | 3 日累计涨幅 ≥ 20% |
| 炸板率警戒 | ≥ 40% |
| 流动性底线 | 日均成交额 ≥ 5000 万，换手率 ≥ 2%，流通市值 ≥ 20 亿 |

## 11. 最终原则

```text
少做，做核心。
看不懂，不做。
没有主线，不做。
没有买点，不做。
没有止损，不做。
仓位算不清，不做。
错了不走，暂停。
```
