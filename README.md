## 核心原则

```text
逻辑定方向
资金验真伪
情绪提弹性
量价定买点
位置看赔率
风控保生存
成本先估算
复盘做迭代
```

一笔交易成立，至少要同时满足：

```text
1. 市场环境允许交易
2. 题材逻辑契合当前主线
3. 板块有资金持续流入
4. 个股是板块内强者
5. 买点位置赔率合理
6. 错了有明确退出点
7. 风险预算和交易成本已提前算清
```

第一原则：

```text
先判断市场能不能做，再判断个股值不值得做。
```
