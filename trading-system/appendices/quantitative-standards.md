# 量化参考标准速查表

## 1. 成交量相关

| 信号 | 量化标准 |
|------|----------|
| 显著放量 | 当日成交额 ≥ 5 日均量的 1.5 倍 |
| 强放量 | 当日成交额 ≥ 5 日均量的 2 倍 |
| 缩量回调 | 回调日成交额 ≤ 前一日成交额的 70%，且 ≤ 5 日均量的 80% |
| 无量上涨 | 上涨日成交额 < 5 日均量的 90% |
| 放量不能上涨 | 成交额 ≥ 5 日均量 1.3 倍，但收盘涨幅 < 1% 或收阴 |
| 板块持续放量 | 板块成交额连续 2 日 ≥ 板块 5 日均量的 1.3 倍 |
| 市场放量 | 全市场成交额 ≥ 近 5 日均值的 1.2 倍 |
| 市场缩量 | 全市场成交额 < 近 5 日均值的 0.85 倍（连续 3 日为空仓信号） |

## 2. 情绪相关

| 信号 | 量化标准 |
|------|----------|
| 涨停数量健康 | 非 ST 涨停 ≥ 30 只 |
| 炸板率高 | 炸板率 ≥ 40% |
| 炸板率可控 | 炸板率 ≤ 25% |
| 跌停增加 | 跌停数较前日增加 ≥ 50% |
| 跌停警戒 | 非 ST 跌停 ≥ 20 只 |
| 连板高度正常 | 最高连板 ≥ 3 板 |
| 板块高潮 | 板块内涨停 ≥ 8 只，或板块涨幅 ≥ 4%，或板块成交额创近 5 日新高 |
| 情绪扩散 | 板块内涨停数较前日增加 ≥ 3 只 |
| 高标杀跌 | 市场最高标跌停或大跌 ≥ 7% |
| 赚钱效应 | 昨日涨停指数今日收阳，且昨日涨停股今日平均涨幅 ≥ 1% |

## 3. 个股相关

| 信号 | 量化标准 |
|------|----------|
| 突破确认 | 收盘价 ≥ 突破位 × 1.005 |
| 突破失败 | 收盘价 < 突破位 × 0.99 |
| 创新高 | 创近 20 日新高 |
| 缩量回踩确认 | 回调日成交额 ≤ 前一日 70% + 次日收阳放量（≥ 回调日 1.2 倍） |
| 放量长阴 | 跌幅 ≥ 5% 且成交额 ≥ 5 日均量 1.5 倍 |
| 冲高回落 | 长上影 ≥ 实体 2 倍 |
| 高位滞涨 | 距近 20 日高点 > 15%，或爆量（≥ 2 倍均量）但涨幅 < 1% |
| 修复确认 | 分歧后 2 日内收复分歧日跌幅 60% 以上 |
| 抗跌确认 | 个股回调幅度 ≤ 板块回调幅度的 70% |
| 流动性底线 | 日均成交额 ≥ 5000 万 + 日均换手率 ≥ 2% + 流通市值 ≥ 20 亿 |

## 4. 指数与技术相关

| 信号 | 量化标准 |
|------|----------|
| 指数强势 | 站上 20 日均线，且 20 日均线向上 |
| 指数震荡 | 在 20 日均线 ±1% 内连续 3 日，均线走平 |
| 指数弱势 | 在 20 日均线下方，或均线向下 |
| 指数破位 | 单日跌 ≥ 2% 且成交额 ≥ 5 日均量 1.3 倍 |
| 多头排列 | 5 日线 > 10 日线 > 20 日线 |
| 三线之上 | 股价 > 5 日线 > 10 日线 > 20 日线 |
| 跌破趋势 | 收盘价 < 5 日线 / 10 日线 / 前低（任一确认） |
| 反弹无量 | 反弹日成交额 < 5 日均量的 90% |
