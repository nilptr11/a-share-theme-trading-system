# 交易模式草案：逻辑、资金、情绪、量价、风控

> 说明：本文是交易系统框架，不构成投资建议。目标是把交易从“凭感觉”变成“有条件、有边界、有复盘”的流程。

## 1. 核心原则

这套模式的核心不是预测，而是先判断市场主线，再跟随已经出现的资金行为。

```text
逻辑定方向
资金验真伪
情绪提弹性
量价定买点
位置看赔率
风控保生存
复盘做迭代
```

一笔交易成立，至少要同时满足：

```text
1. 市场环境允许交易
2. 题材逻辑契合当前主线
3. 板块有资金持续流入
4. 个股是板块内强者
5. 买点位置赔率合理
6. 错了有明确退出点
```

如果只满足“方向看起来对”，但没有资金验证、买点、仓位和止损规则，这不属于有效交易模式。

---

## 2. 适用市场环境

这套模式更适合：

```text
强势市场
结构性行情
题材活跃期
成交量放大期
短线赚钱效应较强阶段
```

不适合：

```text
缩量阴跌
题材一日游
连板高度下降
炸板率高
跌停数量增加
指数放量破位
主线持续退潮
```

第一原则：

```text
先判断市场能不能做，再判断个股值不值得做。
```

## 3. 市场环境评分

每天交易前先给市场打分，满分 10 分。

### 3.1 指数环境：0-2 分

```text
0分：指数破位下跌，均线空头
1分：指数震荡，无明显方向
2分：指数趋势向上，或者至少不拖累情绪
```

### 3.2 成交量环境：0-2 分

```text
0分：市场明显缩量
1分：成交量稳定
2分：成交量持续放大
```

### 3.3 短线情绪：0-3 分

观察：

```text
涨停数量
连板高度
炸板率
跌停数量
昨日涨停表现
高标股表现
```

评分：

```text
0分：跌停多、炸板高、高标杀跌
1分：情绪弱修复
2分：赚钱效应较好
3分：连板高度打开，强势股持续赚钱
```

### 3.4 主线清晰度：0-3 分

```text
0分：没有主线，资金乱打
1分：题材多但持续性弱
2分：有明确主线
3分：主线持续加强，核心股不断超预期
```

### 3.5 市场结论

```text
8-10分：可积极交易
6-7分：轻仓试错
4-5分：谨慎，只做最强
0-3分：空仓或极低仓
```

## 4. 逻辑-资金-情绪三位一体

选股不能只看图形，也不能只看消息。真正值得参与的机会，通常来自三类共振。

### 4.1 逻辑共振：定方向

逻辑解决的是“这个方向有没有想象空间”。重点看：

```text
国家大势
政策支持
产业趋势
业绩兑现
技术进步
事件催化
```

题材优先级可以粗略排序为：

```text
1级：国家大势
2级：政策支持
3级：技术进步
4级：个股利好
```

但短线交易不能只看题材级别，还要看资金反应：

```text
长期看题材级别，短期看资金反应。
```

国家大势不代表马上上涨，个股利好也不一定级别低。真正有效的排序应是：

```text
题材级别 × 资金强度 × 市场阶段 × 个股地位
```

### 4.2 资金共振：验真伪

资金解决的是“市场是否真的认可这个逻辑”。不要过度依赖软件上的主力净流入，更要看真实成交结果。

优先观察：

```text
板块成交额是否持续放大
核心股成交额是否进入市场前排
上涨是否放量
回调是否缩量
分歧后是否快速修复
板块内是否有多个个股联动
```

资金共振成立时，通常不是单只股票独涨，而是：

```text
龙头走强
中军放量
后排扩散
板块成交额放大
分歧后资金回流
```

### 4.3 情绪共振：提弹性

情绪解决的是“行情有没有短期爆发力”。重点看：

```text
题材是否契合市场主旋律
涨停数量是否增加
连板高度是否打开
板块是否集体异动
后排是否扩散
炸板率是否可控
跌停数量是否没有明显增加
```

情绪共振越强，短线弹性越大；但情绪高潮后，也越容易分化。

### 4.4 三位一体判断

理想机会是：

```text
逻辑有级别
资金有验证
情绪有扩散
位置有赔率
风险可控制
```

如果只有逻辑，没有资金，不买；如果只有资金，没有逻辑，谨慎；如果只有情绪，没有位置，容易追高。

## 5. 选股池规则

不全市场乱看，只关注逻辑清晰、资金集中、情绪活跃的地方。

筛选分三层：

```text
底层：契合大势与主线，匹配全年或阶段性市场旋律
中层：资金认可 + 题材容量，能容纳持续资金进出
表层：多逻辑共振 + 高弹性，至少具备2个核心催化点
```

交易化筛选顺序：

```text
1. 方向是不是市场主线？
2. 板块有没有成交额和持续性？
3. 个股是不是板块核心？
4. 当前位置有没有赔率？
5. 错了能不能快速退出？
```

只有前三条成立，是选股；五条都成立，才是交易机会。

### 5.1 板块核心股

优先看：

```text
板块成交额前排
板块涨幅前排
最先启动
回调最抗跌
反弹最主动
能带动板块情绪
```

### 5.2 成交额前排股

重点观察：

```text
全市场成交额前50
板块内成交额前5
当日成交额显著放大的个股
```

成交额太小的个股，即使涨得快，也容易出现流动性风险。

### 5.3 创新高或接近新高的强势股

强势股常见特征：

```text
不断新高
回调不深
修复很快
大盘调整时抗跌
板块修复时领涨
```

不要天然害怕新高。短线里，新高有时不是风险，而是强势确认。

### 5.4 主线第一次分歧后的修复股

条件：

```text
板块主线没有结束
龙头没有彻底走坏
第一次明显分歧
分歧后资金快速回流
核心股率先修复
```

## 6. 个股评分模型

满分 10 分。

```text
成交量：0-3分
相对强度：0-3分
趋势位置：0-3分
市场环境：0-1分
```

### 6.1 成交量

```text
0分：无量
1分：略放量
2分：明显放量
3分：放量且价格强
```

有效放量参考：

```text
当日成交额 ≥ 最近5日均量的1.5倍
强放量：当日成交额 ≥ 最近5日均量的2倍
```

### 6.2 相对强度

```text
0分：弱于大盘和板块
1分：跟随板块
2分：强于板块
3分：板块核心/市场前排
```

判断标准：

```text
大盘跌，它不跌
板块回调，它抗跌
板块反弹，它先涨
同题材里，它涨幅最大、回撤最小、修复最快
```

### 6.3 趋势位置

```text
0分：高位滞涨或破位
1分：位置偏高，赔率一般
2分：趋势健康
3分：低位突破/主升第一次分歧
```

### 6.4 交易门槛

```text
5分以下：不做
6分：只观察或极小仓试错
7分以上：可考虑交易
8-9分：重点关注
10分：极少出现，仍需按风控执行
```

## 7. 买点模型

买点分为三类介入方式：

```text
埋伏低吸：题材未完全启动，适合小仓试错
追涨买入：题材共振明确，只追前排核心
回踩加仓：核心逻辑未破，只在重新转强后加仓
```

三类介入方式的风险不同：

```text
埋伏低吸赔率高，但确定性低，不可重仓
追涨买入确定性更高，但容易追在高潮，不买后排
回踩加仓看似安全，但不能把加仓做成下跌补仓
```

核心原则：

```text
买点必须同时回答：为什么是它、为什么是现在、错了在哪里走。
```

### 7.1 买点一：低位放量突破

适合题材刚启动。

条件：

```text
1. 个股经过一段时间横盘
2. 题材已有预热或板块已有异动迹象
3. 突破近期平台高点
4. 成交额明显放大
5. 收盘站上突破位
6. 板块同步走强
7. 大盘环境不差
```

理想形态：

```text
横盘整理 → 放量突破 → 收盘强势
```

买入方式：

```text
突破确认时小仓试错
收盘站稳可加一点
次日强势确认再加
```

失败信号：

```text
突破后第二天低开低走
跌回平台内部
成交额放大但价格走弱
板块没有跟随
```

处理原则：

```text
突破失败要快速止损，不补仓摊低成本。
```

---

### 7.2 买点二：主升第一次缩量回踩

适合已经走强的趋势股。

条件：

```text
1. 个股已经确认主升趋势
2. 第一次明显回调
3. 回调缩量
4. 不破5日线/10日线/前高平台
5. 板块没有退潮
6. 次日重新转强
```

买入方式：

```text
回踩支撑附近观察
重新放量转强时介入
不在下跌惯性中盲目接
```

核心原则：

```text
买转强，不买下跌惯性。
```

失败信号：

```text
缩量回踩变成放量下跌
跌破关键支撑
反弹无量
板块核心股走坏
```

---

### 7.3 买点三：强势分歧后的反包

适合短线情绪强的阶段。

条件：

```text
1. 个股是板块核心
2. 前一日出现分歧，如炸板、断板、冲高回落
3. 次日没有继续走弱
4. 开盘或盘中快速转强
5. 成交额能承接抛压
6. 板块其他核心股配合
```

买入方式：

```text
反包确认时介入
不能提前赌
```

失败信号：

```text
高开低走
反包失败
放量不能上涨
板块分歧扩大
龙头跌停或大跌
```

## 8. 卖点模型

卖点优先级高于买点。买错不可怕，错了不卖才危险。

卖出信号分四类：

```text
封单异常：涨停承接变弱，可能是炸板或分歧扩大前兆
走势不及预期：该强不强，板块涨它不涨，突破后跌回平台
资金分歧：放量不能上涨，核心股承接变差，板块回流失败
逻辑破位：题材降温、事件落地、政策或业绩不及预期
```

不要只等逻辑破位才卖。逻辑破位往往滞后，价格、资金和情绪经常更早反映变化。

卖出判断可以用组合原则：

```text
逻辑破、资金弱、技术破、情绪退，任意两个出现就减仓或卖出；
触发硬止损，必须卖出。
```

### 8.1 止损卖点

#### 固定亏损止损

```text
单笔亏损达到总本金的1%-3%，必须处理。
```

以 2 万本金为例：

```text
1% = 200元
2% = 400元
3% = 600元
```

如果一笔交易可能亏超过这个范围，说明仓位太重或买点太差。

#### 技术破位止损

出现以下情况要退出：

```text
跌破突破位
跌破前低
跌破5日线/10日线
放量长阴
跌回平台内部
反包失败
```

#### 市场环境止损

即使个股没怎么跌，如果市场环境突然恶化，也要减仓或退出：

```text
高标集体杀跌
主线龙头跌停
指数放量破位
跌停数量明显增加
板块大面积退潮
```

---

### 8.2 止盈卖点

#### 趋势持有

如果个股继续满足：

```text
放量上涨
缩量回调
不破趋势
板块仍强
核心股没有明显走弱
```

可以继续持有。

#### 高潮兑现

出现以下情况要考虑兑现一部分：

```text
连续大涨
板块集体高潮
后排补涨
媒体大量讨论
高位爆量滞涨
封单快速减少
大单频繁撤单
涨停反复打开
```

#### 走弱卖出

出现以下情况要降低预期或卖出：

```text
上涨无量
冲高回落
跌破5日线
反弹不过前高
板块回流它不涨
强势股变成跟风股
```

原则：

```text
强势股一旦不强，就不能继续用强势股逻辑持有。
```

## 9. 仓位规则

高复利不是靠无脑重仓，而是靠在可控风险下持续抓机会。

### 9.1 初始阶段：2万以内

以下仓位适用于通过小仓验证之后；验证期按第 14 章执行更保守的仓位。

建议：

```text
单票20%-30%仓位
最多同时2-3只
单笔最大亏损控制在本金1%-3%
```

以 2 万本金为例：

```text
单票仓位：4000-6000元
单笔最大亏损：200-600元
```

如果单票 6000 元，亏 5%：

```text
6000 × 5% = 300元
```

这个亏损可控。

如果满仓亏 10%：

```text
20000 × 10% = 2000元
```

一次就亏掉本金 10%，会严重破坏复利节奏。

### 9.2 分阶段仓位管理

```text
2万-5万：练系统，轻仓验证
5万-10万：适度放大，控制回撤
10万-30万：重视稳定性
30万以上：优先保住利润
```

原则：

```text
资金越大，越要重视回撤控制。
```

## 10. 空仓规则

以下情况空仓或极低仓：

```text
1. 市场成交量持续萎缩
2. 指数放量破位
3. 连板高度下降
4. 高标股连续被核
5. 跌停数量明显增加
6. 主线题材退潮
7. 板块轮动太快，一日游严重
8. 自己连续亏损3笔以上
9. 情绪失控，想报复交易
10. 看不懂市场主线
```

重要原则：

```text
看不懂的时候，不交易就是交易。
```

---

## 11. 绝对不能买的情况

以下情况禁止买入：

```text
1. 市场评分低于4分还想重仓
2. 只有逻辑，没有资金验证
3. 板块没有持续性，只因个股突然拉升追入
4. 板块高潮日，前排买不到，去买后排
5. 高位爆量滞涨还追涨
6. 个股跌破趋势后幻想反弹
7. 没有止损位就买入
8. 因为亏损想快速回本而买入
9. 因为别人推荐而买入
10. 因为看起来便宜而买入弱股
11. 买入理由只剩“我觉得它会涨”
```

判断标准：

```text
如果不能写出买入逻辑、止损位置、目标情景，就不能买。
```

---

## 12. 必须卖出的情况

以下情况必须处理，不允许幻想：

```text
1. 买入逻辑被证伪
2. 跌破预设止损位
3. 单笔亏损触及本金1%-3%上限
4. 板块主线退潮
5. 龙头股明显走坏
6. 个股放量长阴破位
7. 反弹无量且不过前高
8. 强势股变弱，板块回流它不涨
9. 封单异常且无法回封
10. 资金分歧扩大，放量不能上涨
11. 自己开始找理由硬扛
12. 原计划短线，亏损后改成长线
```

关键纪律：

```text
短线交易不能因为亏损被动变成长线投资。
```

## 13. 标准交易流程

一笔标准交易应按以下流程执行。

### 13.1 盘前

```text
1. 给市场环境打分
2. 确认是否允许交易
3. 找出当前主线板块
4. 判断主线逻辑级别
5. 检查板块资金是否持续放大
6. 判断情绪是否扩散
7. 找出板块核心个股
8. 标记可能买点
9. 预设止损位置
10. 预设仓位
```

### 13.2 盘中

```text
1. 观察板块是否继续放量
2. 观察核心股是否强于板块
3. 观察题材情绪是否扩散或分化
4. 等待标准买点出现
5. 买入前确认止损空间
6. 按计划仓位买入
7. 不临时扩大仓位
```

### 13.3 买入后

```text
1. 判断走势是否符合预期
2. 符合预期则持有
3. 不符合预期则减仓或止损
4. 不因短期浮盈随意加仓
5. 不因短期浮亏随意补仓
```

### 13.4 卖出后

```text
1. 记录买入理由
2. 记录卖出理由
3. 记录是否遵守纪律
4. 记录盈亏
5. 归类错误类型
```

## 14. 小仓验证期执行规则

当前阶段不直接以“2万到100万”为执行目标，而是先验证系统是否在自己手里具备正期望。

### 14.1 验证周期

```text
至少30笔交易
或连续1-3个月
二者满足其一后再做阶段评估
```

验证期目标不是赚最多，而是确认：

```text
规则能不能执行
亏损能不能控制
哪类买点最有效
系统是否有正期望
自己是否适合这种交易节奏
```

### 14.2 验证期仓位

```text
单票10%-20%仓位
最多同时持有2只
单笔最大亏损控制在本金1%以内
不做满仓单吊
不因连续盈利临时放大仓位
```

以 2 万本金为例：

```text
单票仓位：2000-4000元
单笔最大亏损：200元以内
```

验证期宁可收益慢，也不能用大仓位掩盖系统缺陷。

### 14.3 验证期交易门槛

验证期只做符合以下条件的交易：

```text
1. 市场评分 ≥ 6分
2. 个股评分 ≥ 7分
3. 逻辑、资金、情绪至少两项共振
4. 买点属于 7.1-7.3 定义的标准模型，验证期不做“其他”买点
5. 买入前已经写清止损位
6. 单笔最大亏损不超过本金1%
7. 买入理由可以在盘前或盘中明确写出
```

如果市场评分低于 6 分，只观察和记录，不做实盘验证交易。

### 14.4 暂停机制

出现以下情况，立即暂停实盘验证：

```text
连续亏损3笔
单日亏损超过本金2%
账户从阶段高点回撤超过8%
出现一次不按止损执行
出现一次报复交易
连续3笔交易不符合文档规则
```

暂停后只做复盘，不急着把亏损赚回来。

### 14.5 验证通过标准

完成 30 笔交易或 1-3 个月后，满足以下条件，才进入下一阶段：

```text
胜率 ≥ 45%
盈亏比 ≥ 1.5
最大回撤 < 10%
规则执行率 ≥ 90%
没有单笔纪律性大亏
主要盈利来自标准买点，而不是偶然暴利
```

如果不满足，不扩大仓位，只优化规则或继续小仓验证。

### 14.6 验证期记录要求

每笔交易必须记录：

```text
市场评分
逻辑级别
资金状态
情绪状态
个股评分
买点类型
买入仓位
预设止损
实际卖点
盈亏金额
是否遵守系统
错误类型
```

没有记录的交易，视为无效样本。

## 15. 2万本金阶段目标管理

两年从 2 万到 100 万，需要 50 倍，约等于：

```text
月均复利约17.7%
交易日均复利约0.78%
```

资金路径参考：

```text
第0月：2.0万
第3月：3.3万
第6月：5.3万
第9月：8.7万
第12月：14.1万
第15月：23.0万
第18月：37.5万
第21月：61.1万
第24月：100万
```

但实际执行中，不应只盯收益目标，更应先验证系统。

优先目标：

```text
连续1个月严格执行纪律
连续3个月不出现大回撤
连续6个月资金曲线稳定向上
连续12个月跑赢市场且最大回撤可控
```

如果第一年仍无法稳定盈利，不应盲目加仓追赶目标。

## 16. 回撤控制

复利最大敌人是大回撤。

```text
亏10%，需要涨11.1%回本
亏20%，需要涨25%回本
亏30%，需要涨42.9%回本
亏50%，需要涨100%回本
```

阶段性回撤标准：

```text
10%以内：优秀
15%以内：还能修复
20%以上：复利节奏明显受损
30%以上：系统需要暂停检查
```

如果账户从阶段高点回撤超过 15%，应降低仓位。

如果回撤超过 20%，应暂停交易，复盘系统和执行问题。

## 18. 完整交易案例模板

### 案例：低位放量突破

#### 背景

```text
市场评分：8分
指数：震荡向上
成交量：连续放大
短线情绪：涨停数量增加，连板高度打开
主线：AI板块持续走强
```

#### 选股

```text
个股A属于AI板块
逻辑级别：政策支持 + 技术进步，属于多逻辑共振
资金状态：板块成交额连续放大，个股成交额进入板块前5
情绪状态：板块涨停数量增加，核心股带动后排扩散
过去10个交易日横盘整理
今日放量突破平台高点
涨幅强于板块平均水平
```

#### 买入计划

```text
买点类型：低位放量突破
买入条件：盘中突破平台且成交额明显放大
买入仓位：25%
止损位置：跌回平台内部
最大亏损：本金2%以内
```

#### 买入后处理

```text
情况一：次日继续放量上涨，持有
情况二：次日缩量回踩不破突破位，观察
情况三：次日低开低走跌回平台，止损
情况四：板块整体退潮，主动减仓
```

#### 卖出计划

```text
趋势强：持有到跌破5日线或放量滞涨
板块高潮：先兑现一部分
技术破位：全部卖出
```

#### 复盘问题

```text
买入是否符合系统？
止损是否提前设定？
仓位是否超标？
卖出是否按规则执行？
这笔交易能否重复？
```

## 19. 最终交易闭环

```text
市场环境判断
        ↓
确定是否交易
        ↓
寻找主线板块
        ↓
判断逻辑级别
        ↓
验证资金共识
        ↓
观察情绪扩散
        ↓
筛选核心个股
        ↓
等待标准买点
        ↓
控制仓位买入
        ↓
按规则止损/止盈
        ↓
每日复盘优化
```

这套模式真正要解决的不是“每次都买到涨停”，而是：

```text
在市场给机会时敢做；
在市场不给机会时不做；
做错时亏小钱；
做对时尽量拿住；
长期让盈利交易覆盖亏损交易。
```
